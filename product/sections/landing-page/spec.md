# Landing Page Specification

## Overview
A marketing landing page that introduces HomeStaff to potential users. It features a headline-focused hero section, showcases product features and the problems it solves, displays testimonials, and highlights the current "Free for Limited Time" offer with a prominent sticky CTA bar.

## User Flows
- Visitor lands on page → Scrolls through content → Clicks "Get Started" CTA → Redirects to registration
- Visitor clicks navigation links → Smooth-scrolls to corresponding section
- Visitor clicks Login → Redirects to login page

## UI Requirements
- **Navbar:** Full navigation with logo, section anchor links (Features, Testimonials, Pricing), Login and Sign Up buttons
- **Hero Section:** Large headline, supporting subtext, prominent "Get Started" CTA button
- **Problem/Solution Section:** Highlights pain points and how HomeStaff addresses them
- **Features Overview:** Grid or cards showcasing key product capabilities
- **Testimonials:** User quotes with names/avatars
- **Pricing Section:** Single promotional banner — "Free for All for Limited Time"
- **Sticky CTA Bar:** Persistent bar at bottom with sign-up button visible while scrolling
- **Footer:** Company info, social media links (Twitter, Facebook, LinkedIn), legal/support links (Terms, Privacy, Contact)
